## Technical Production Notes

### Visual Assets Required:
1. **Logo/Branding**: Medical cross with lock icon
2. **Color Scheme**:
   - Primary: Blue (#2563EB) - Trust, Healthcare
   - Secondary: Green (#10B981) - Security, Success
   - Accent: Purple (#8B5CF6) - Encryption, Innovation
3. **Icons**: Lock, Medical Cross, Shield, Key, Document
4. **Transitions**: Smooth fades, slide animations (0.3s duration)
5. **Background**: Clean white or subtle gradient

### Screen Recording Sections:
1. **Terminal Demo** (0:30-0:44):
   - Record Hardhat console interaction
   - Use syntax highlighting for code
   - Show actual function calls and outputs

2. **Test Results** (0:44-0:52):
   - Record `npm run test` execution
   - Highlight passing tests
   - Show coverage report if available

3. **File Structure** (0:52-0:56):
   - Quick folder tree view
   - Highlight: contracts/, test/, scripts/, README.md

### Text Styling:
- **Titles**: Bold, 36-48pt, Center-aligned
- **Feature Text**: Regular, 24-30pt, Left-aligned
- **Code Snippets**: Monospace, 18-20pt
- **Bullet Points**: Check marks (‚úì), X marks (‚ùå)

### Audio/Music:
- **Background Music**: Subtle, professional, tech-focused (royalty-free)
- **Volume**: Keep low (20-30%) to not overpower narration
- **Style**: Ambient, modern, uplifting

### Pacing:
- **Cuts**: Every 4-6 seconds maximum
- **Animations**: Quick (0.2-0.4s), professional
- **Text Display**: 2-3 seconds minimum for readability

### Quality Settings:
- **Resolution**: 1920x1080 (1080p) minimum
- **Frame Rate**: 30 fps or 60 fps
- **Export Format**: MP4 (H.264)
- **Bitrate**: 8-12 Mbps
